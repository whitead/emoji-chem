## Install

```sh
pip install emojichem
```

## Quickstart

```py
import emojichem
emojichem.emoji_draw('CCN(CC)C(=O)[C@H]1CN([C@@H]2Cc3c[nH]c4c3c(ccc4)C2=C1)C')
```
